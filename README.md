# Professional CV | Kassahun Tigabu Teshome

Europass-style, ATS-friendly CV website — HTML, CSS, and vanilla JavaScript. No frameworks.

## Features

- **International layout** — Europass-inspired, suitable for EU/UN/NGO and global tech roles  
- **Responsive** — One-column on mobile, clean layout on desktop  
- **Print & PDF** — Print CV and download as PDF from the browser  
- **Dark / Light mode** — Toggle with preference saved in `localStorage`  
- **Accessible** — Semantic HTML5, skip link, ARIA where needed  

## Run locally

Open in a browser:

```bash
# From project root
start cv/index.html
```

Or serve with a local server (e.g. Python):

```bash
cd cv
python -m http.server 8080
# Open http://localhost:8080
```

## Structure

```
cv/
├── index.html    # CV content & structure
├── style.css     # Layout, theme, print styles
├── script.js     # Print, PDF, theme toggle
└── Photo/        # Profile photo
```
